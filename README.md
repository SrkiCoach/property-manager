# Property Manager

Full-stack business application for managing customers, properties, bookings and payments.

## Technology stack

### Backend

- Java 21
- Spring Boot 4.1
- Spring Data JPA
- Bean Validation
- H2
- Maven

### Frontend

- Angular 21
- PrimeNG
- ngx-translate
- Reactive Forms
- SCSS

## Implemented features

### Customers

- Create customer
- Edit customer
- Delete customer
- Confirmation dialog
- Client-side validation
- Server-side validation
- Structured API errors
- Server-side pagination
- Server-side sorting
- Search by first name, last name, email and phone
- English and Greek translations
- Toast notifications
- Integration tests
